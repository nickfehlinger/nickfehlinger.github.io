Work

GIGANTIC

github

GIGANTIC is a scheduling and budgeting app for freelancers and contractors.

    Key Features:
  • Users enter gigs into the calendar on their dashboard which is powered by full_calendar.
  • GIGANTIC pulls database information as a jSON array and dynamically calculates information with JavaScript to inform the user if they are scheduled to earn enough to make their budget.
  • Built with:
    • Ruby On Rails
    • JavaScript
    • jQuery
    • jSON
    • Bootstrap v3.3.7
    • full_calendar
    • Google Maps API
    • simple_form
    • Bootstrap 3 Datetime Picker
    • Bootstrap Tour
    • Charts.js
    • Devise

Fleet

github

Fleet is a group planning tool for travellers.

    Key Features:
  • Build a group and add your friends. Fleet uses has_friendship to build relationships between users.
  • Suggest destinations and add events to itineraries. Fleet finds the coordinates of the city you chose using geocoder and then feeds them through the Acehopper API to find events in the area.
  • Built with:
    • Ruby On Rails
    • JavaScript
    • jQuery
    • Bootstrap v3.3.7
    • full_calendar
    • Acehopper API
    • Devise
    • has_friendship
    • jSON
    • Geocoder
    • Paperclip

Scratch

github

Scratch is a cooking blog that encourages users to avoid pre-made foods.

    Key Features:
  • Users have full CRUD capabilities on posts and comments that they create themselves. They can also like posts.
  • The front page is populated with the most popular posts on the site based on likes. Users can also sort by category or search for posts.
  • Built with:
    • Ruby On Rails
    • JavaScript
    • Bootstrap v3.3.7
    • Bootstrap Carousel
    • Devise

Float Like A Butterfly Sting Like A Jukebox

github

Float Like A Butterfly Sting Like A Jukebox is a JavaScript jukebox that plays 10 different covers and mashups of Crazytown's Butterfly

    Key Features:
  • Play, Pause, Stop, Forward, and Backward buttons control the player.
  • Pulls music through a jSON request of Soundcloud's API.
  • Built with:
    • JavaScript
    • Soundcloud API
    • jSON
